Introduction to Web Mapping and GeoServer

Web mapping is a technology used to create interactive, web-based maps. GeoServer is an open-source server software designed for sharing, processing, and editing geospatial data. It allows you to serve maps and data from various spatial data sources.

Choropleth Map

A choropleth map is a thematic map in which areas are shaded or patterned in proportion to the value of a particular variable being represented.
